[Personal Project] KBBQ Idle WebGL 리뉴얼 공개

Unity 2022.3 LTS로 만든 KBBQ Idle 프로젝트를 WebGL 기준으로 다시 다듬었습니다.

이번 리뉴얼에서 집중한 부분:
- 4-그릴 조리 루프 강화: `구매 -> 올리기 -> 뒤집기 -> 수거 -> 서빙`
- 손님 큐 시각화 개선: 도트 캐릭터 + 주문 말풍선 + 요청 고기 아이콘
- 업그레이드 UX 개선: 리스트 클릭 시 모달 상세/구매, 단계별 그릴 비주얼 변화
- 서빙 피드백 강화: 손님 식사 리액션 애니메이션
- 몰입감 보강: 지글지글 레이어 사운드(루프 + crackle)
- WebGL/모바일 대응: 레이아웃/가독성/운영 페이지 정리
