KBBQ Idle (WebGL)

This GitHub Pages site is reserved for the WebGL build output. The build artifacts are generated locally from Unity to keep this repo lightweight.

To generate the playable build:

  1. Open the Unity project (2022.3.62f3)
  2. Run KBBQ/Build WebGL (docs) in the Editor
  3. Commit the generated docs/ output and push to main

See the repo README for details.
