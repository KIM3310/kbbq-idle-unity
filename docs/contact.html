Contact & Support

Primary support channel: GitHub Issues. This route is used for gameplay bugs, deployment failures, and data integrity incidents.

Bug Report Checklist

  • Page URL and browser/device details.
  • Expected behavior vs actual behavior.
  • Exact timestamp and screenshot/video if possible.
  • Current build name shown on Play page (for example KBBQIdleWebGL).

Operational Support Scope

  • WebGL load failures and broken build output mapping.
  • Gameplay progression/save inconsistencies.
  • Cloudflare Pages deployment checks and release validation.

Response Notes

Support responses are best-effort. High-impact issues (playability/data integrity) are prioritized over visual enhancement requests.

For urgent regressions, include [REGRESSION] in the issue title.
